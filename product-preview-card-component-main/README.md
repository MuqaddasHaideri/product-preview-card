
# Product Preview Card Component

A clean, responsive product preview card component built with a **mobile-first workflow**. This project focuses on high-quality CSS styling, layout management, and matching a specific design brief with precision.

---

## Built With

This project utilizes modern web standards to ensure a lightweight and accessible user experience:

*   **HTML5:** Semantic markup used for better accessibility and SEO.
*   **CSS3:** Leverages custom properties (variables), Flexbox, and Media Queries.
*   **Google Fonts:** Specialized typography for a premium, high-end aesthetic.
*   **Mobile-First Workflow:** Optimized for small screens first before scaling up to desktop.

---

## Key Features

*   **Fully Responsive:** Seamlessly transitions between mobile and desktop layouts for any device.
*   **Interactive States:** Smooth hover and active states on buttons to improve user engagement.
*   **Optimized Assets:** Uses adaptive image loading to ensure fast performance and proper framing.
*   **Aesthetic UI:** Focuses on clean spacing, consistent padding/margins, and a minimalist color palette.

---

##  Technical Focus

During development, I prioritized the following technical aspects:

### 1. Flexbox Mastery
I used Flexbox to manage the alignment of the text content and the overall card structure, ensuring the layout remains stable across different screen sizes.

### 2. Typography & Design
I carefully balanced font weights, colors, and letter spacing to match the "luxury" feel required for a high-end product description.

### 3. Responsive Images
Implemented the `<picture>` tag and modern CSS techniques to ensure the browser serves the correct image version (mobile vs. desktop) based on the user's viewport.
